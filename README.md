## Delivery API

A RESTful API for managing deliveries and user authentication using JWT.

Built with:
- TypeScript
- Node.js
- Express
- JSON Web Tokens (JWT)
- PostgreSQL
- Prisma ORM
- Zod (schema validation)
- Docker (database container)
- Jest (automated tests)
- Insomnia (API testing)

This repository was created for learning purposes as part of the Rocketseat Node.js course.
